# Vendor Management - UI Design Specification

## ğŸ“‹ Executive Summary

The Vendor Management module in HRMS provides an interface to manage staffing vendors/intermediaries in C2C (Corp-to-Corp) arrangements. Vendors are companies in the vendor chain between the staffing agency and the end client.

**Data Storage:** HRMS Database
- Vendors â†’ `hrms_vendors` table in HRMS
- Project-Vendor assignments â†’ `hrms_project_vendors` table

**Key Feature:** When a vendor is identified as "next to client" (vendor_level = 1), the system can automatically sync vendor information to CRM as a client with contacts.

**Page URLs:**
- List: `/hrms/vendors`
- Detail: `/hrms/vendors/:vendorId`
- Create: `/hrms/vendors/new`
- Edit: `/hrms/vendors/:vendorId/edit`

**Access Level:** HRMS Admin, HR Manager, Project Manager  
**Primary Function:** Manage vendors for employee project assignments and C2C billing

### 4.2 Sync Process

1. **Check Existing Client:** Look for existing CRM client with same name or code
2. **Create/Update Client:** 
   - If no match: Create new client in CRM
   - If match found: Update existing client details
3. **Sync Contacts:**
   - Create contacts in CRM with `contact_type = 'Client Empanelment'`
   - Link contacts to client via `client_id`
4. **Update Vendor Record:**
   - Set `is_direct_to_client = true` on `hrms_project_vendors`
   - Store CRM `client_id` reference for future sync

### 11.1 Form Validation

| Field | Rules |
|-------|-------|
| Vendor Name | Required, max 255 chars, unique per tenant |
| Vendor Code | Required, max 50 chars, unique per tenant, alphanumeric |
| EIN | Optional, format XX-XXXXXXX |
| Email | Valid email format |
| Phone | Valid phone format |
| Payment Terms | Positive integer, max 365 |
| Markup % | 0-100 |
| Bill Rates | Positive decimal, max 2 decimal places |

### 11.2 Business Rules

1. **Vendor Code Uniqueness:** Must be unique within tenant
2. **Deactivation:** Cannot deactivate vendor with active projects
3. **Sync to Client:** Only vendors at level 1 (direct to client) can be synced
4. **EIN Masking:** Display only last 4 digits in list view

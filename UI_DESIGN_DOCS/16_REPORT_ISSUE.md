## âœ… 6. Validation Rules

### Issue Submission Validations

| Field | Validation | Error Message |
|-------|------------|---------------|
| Type | Required | "Please select an issue type" |
| Severity | Required | "Please select a severity level" |
| Title | Required, 10-150 chars | "Title must be between 10 and 150 characters" |
| Description | Required, 50-2000 chars | "Description must be at least 50 characters" |
| Steps | Optional, max 1000 chars | "Steps to reproduce cannot exceed 1000 characters" |
| Screenshots | Max 5MB each, 5 files max | "File exceeds maximum size of 5MB" |
| File Types | Images, PDFs, logs | "Only images, PDFs, and log files are allowed" |

### Admin Response Validations

| Field | Validation | Error Message |
|-------|------------|---------------|
| Response | Required for resolution, min 20 chars | "Response must be at least 20 characters" |
| Status | Required | "Please select a status" |
| Internal Notes | Optional, max 2000 chars | "Internal notes cannot exceed 2000 characters" |

# Employee Projects - UI Design Specification

## ğŸ“‹ Executive Summary

The Employee Projects module manages all project assignments for employees, including complex vendor chains, rate management, LCA tracking, and project documents (MSA/PO/COI). It supports temporal rate history tracking for rate changes over time.

**Page URLs:**
- List: `/hrms/projects`
- Detail: `/hrms/projects/:projectId`
- Create: `/hrms/projects/new`
- Edit: `/hrms/projects/:projectId/edit`

**Access Level:** HRMS Admin, HR Manager, Project Manager  
**Primary Function:** Complete project lifecycle management with vendor chain visualization

**Duplicate Client Detection:**
- System checks for existing clients as user types in Client Name or Client Code
- Fuzzy matching on client name (similarity threshold: 80%)
- Exact match on client code
- If potential duplicate found:
  - Shows warning with existing client details
  - **[Use Existing]** - Selects the existing client instead
  - **[Ignore]** - Proceeds with creating new client (user confirms it's different)
- Prevents exact duplicate client codes (validation error)

**Vendor Selection Behavior:**
- Searchable dropdown lookups from `hrms_vendors` table
- Selecting a vendor auto-populates contact fields from vendor master data
- Contact fields can be overridden for project-specific contacts
- "Create New Vendor" opens inline form to add new vendor to master table
- New vendors are immediately available in the dropdown after creation

**Duplicate Vendor Detection (when creating new):**
- System checks for existing vendors as user types Vendor Name or Vendor Code
- Fuzzy matching on vendor name (similarity threshold: 80%)
- Exact match on vendor code
- If potential duplicate found:
  - Shows warning: "âš ï¸ Similar vendor found: 'ATVS Corporation' (ATVS) [Use Existing] [Ignore]"
  - **[Use Existing]** - Selects the existing vendor from the dropdown
  - **[Ignore]** - Proceeds with creating new vendor (user confirms it's different)
- Prevents exact duplicate vendor codes (validation error: "Vendor code already exists")

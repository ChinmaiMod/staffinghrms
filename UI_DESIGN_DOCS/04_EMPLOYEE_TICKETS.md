# Employee Tickets - UI Design Specification

## ğŸ“‹ Executive Summary

The Employee Tickets module provides a comprehensive helpdesk system where employees can submit requests/tickets to HR or Immigration teams. It includes self-service ticket creation for employees and a management interface for HR/Immigration teams to review, respond, and resolve tickets.

**Employee Self-Service URL:** `/hrms/my-tickets` (Employee Portal)  
**Admin Tickets URL:** `/hrms/tickets`  
**Access Level:** 
- **Employee Portal:** All employees (self-service)
- **HR Tab:** HR Team members
- **Immigration Tab:** Immigration Team members

**Primary Functions:**
- Employee self-service ticket creation
- Department-based ticket routing (HR vs Immigration)
- Status tracking with full history
- Comments and attachments
- Email notifications via business-specific Resend API

**Ticket Number Format:** `<business_short_name>TKT<4-digit-seq>` (e.g., `IESTKT0042` for Intuites LLC)

// Email sending logic based on business
async function sendTicketEmail(ticket, templateKey, recipientEmail) {
  // Get business-specific Resend API key
  const business = await getBusinessById(ticket.business_id);
  const resendApiKey = business.resend_api_key;
  
  // Get email template
  const template = await getEmailTemplate(templateKey, ticket.tenant_id);
  
  // Build email with template variables
  const emailContent = buildEmailContent(template, {
    ticket_number: ticket.ticket_number,
    subject: ticket.subject,
    status: ticket.status,
    employee_name: ticket.employee_name,
    department: ticket.department,
    ...
  });
  
  // Send via Resend with business-specific API
  const resend = new Resend(resendApiKey);
  await resend.emails.send({
    from: business.email_from_address,
    to: recipientEmail,
    subject: emailContent.subject,
    html: emailContent.body,
  });
}

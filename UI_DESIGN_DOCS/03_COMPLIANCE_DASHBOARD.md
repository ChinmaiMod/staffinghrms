# Compliance Dashboard & Manager - UI Design Specification

## ğŸ“‹ Executive Summary

The Compliance module provides visibility and management of all compliance-related items across the organization. It includes a dashboard for high-level overview and a detailed manager for tracking, resolving, and auditing compliance items.

**Page URLs:**
- Dashboard: `/hrms/compliance`
- Manager: `/hrms/compliance/manage`
- Calendar: `/hrms/compliance/calendar`
- Item Detail: `/hrms/compliance/:complianceId`

**Access Level:** HRMS Admin, Compliance Officer, HR Manager  
**Primary Function:** Track document expirations, visa renewals, and regulatory compliance

**Security & Access Control:**
- Compliance items often include sensitive personal and immigration data; access must be restricted by RBAC and tenant isolation (RLS).
- Bulk actions and status changes (e.g., mark complete, snooze) must be enforced server-side; UI-only restrictions are insufficient.
- Exports/reports containing PII should require appropriate permission and may require â€œrecent authenticationâ€.
